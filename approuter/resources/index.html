App BTP - Object Store
  • GET /v1/ListObjects request that provides List of Objects present in the provided S3 Bucket.
  • GET /v1/ListVersions request that provides List of Objects present in the provided S3 Bucket with Versions.
  • GET /v1/BucketVersionHandler Method to modify Bucket versioning Status
  • GET /v1/BucketVersionStatus request to show Bucket versioning Status
  • GET /v1/GetObject request to get object from the AWS S3 Bucket
  • GET /v1/PutObject request to put object in the AWS S3 Bucket
  • GET /v1/DeleteObject request to delete object in the AWS S3 Bucket
  • GET /v1/DeleteObjectVersion request to delete object version in the AWS S3 Bucket
